---
- File [`orbi_iimn_gnps_quant.csv`](https://github.com/plyush1993/MetaboCensoR/blob/main/Input_Examples/orbi_iimn_gnps_quant.csv) is *Example Dataset* in the App. It is the LC-MS profiling dataset, described in the study [[1]](https://pubs.acs.org/doi/10.1021/acs.analchem.4c05577).<br>
Briefly, methanol extracts from the plant ashwagandha [*Withania somnifera* (L.) Dunal] together with blanks were analyzed on Thermo Q-Exactive Plus Orbitrap in DDA positive mode. Raw mzML files were then processed in mzMine in default pre-settings for UPLC-DDA.

## Peak table (`.csv`)

- `xcms` table output was generated as follows:
```r
library(xcms)

pd <- data.frame(...)
xdata <- readMsExperiment(...) or readMSData(...)

xdata <- findChromPeaks(...)
xdata <- adjustRtime(...)
xdata <- groupChromPeaks(...)
xdata <- fillChromPeaks(...)

fvals <- featureValues(xdata, value = "into")
fdef <- featureDefinitions(mse)[, c("mzmed", "mzmin", "mzmax", "rtmed", "rtmin", "rtmax")]
fvals <- cbind(Row.names = rownames(fdef), fdef, fvals)

write.csv(fvals, "... .csv", row.names = F)
```
---
- `mzMine` table output was generated by the module *`Export molecular network files; Options: no normalization, Intensity - Area, CSV export - Simple`*. File ends with *`_quant.csv`*.
---
- `MS-Dial` table output was generated by the module *`Export -> Alignment result -> Selected Options: Raw data (Area); format csv; trim content for Excel`*.
---

## Spectra file (`.mgf`)

- `xcms` spectra output was generated as follows:
```r
library(xcms)
library(Spectra)
library(MsBackendMgf)

pd <- data.frame(...)
xdata <- readMsExperiment(...) or readMSData(...)

xdata <- findChromPeaks(...)
xdata <- adjustRtime(...)
xdata <- groupChromPeaks(...)
xdata <- fillChromPeaks(...)

ms2 <- featureSpectra(xdata)
ms2_cons <- combineSpectra(ms2, f = ms2$feature_id,
                           p = rep(1, length(ms2)),
                           peaks = "intersect",
                           ppm = 10, minProp = 0.75)
ms2_cons <- ms2_cons[lengths(ms2_cons) > 1]
source("https://raw.githubusercontent.com/jorainer/xcms-gnps-tools/master/customFunctions.R")
ms2_cons <- formatSpectraForGNPS(ms2_cons)
export(ms2_cons, backend = MsBackendMgf(),
       file = "... .mgf")
```
---
- `mzMine` spectra output for GNPS was generated by the module *`Export molecular network files; Options: Filter rows - MS2 or Ion Identity; Merged (simple); single scan: Merged across energies; merging m/z tolerance: 0.001 m/z or 5 ppm`*.
- `mzMine` spectra output for SIRIUS was generated by the module *`Export for SIRIUS; Options: no normalization; Merged (simple); single scan: Merged across energies; merging m/z tolerance: 0.001 m/z or 5 ppm; m/z tolerance: 0.003 m/z or 5 ppm`*.
---
- `MS-Dial` spectra output was generated by the module *`Export -> Alignment result -> GNPS -> Selected Options: GNPS2 or GNPS(MSDIAL4 format)`*.
